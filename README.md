# 📝 Registration Form

A detailed **Registration Form** built using **HTML** and **CSS** with a green border theme and background image.

## 📁 Project Structure

```
project/
│
├── register.html              # Registration form page
│
└── assets/
    ├── css/
    │   └── style.css       # Main stylesheet
    └── img/
        └── register.png    # Background image
```

## 🚀 Features

- First Name & Last Name fields
- Email Address & Phone Number fields
- Password & Confirm Password fields
- Gender dropdown (Male, Female, Other)
- Date of Birth date picker
- Address textarea
- Languages checkboxes (Tamil, English, Hindi, French, Spanish, Chinese, Japanese)
- Register submit button
- Green border card with box shadow
- Custom background image

## 🛠️ Technologies Used

- HTML5
- CSS

## ⚙️ Setup & Usage

1. **Clone the repository**
```bash
   git clone https://github.com/Monisha71326/register-page.git
```

2. **Navigate to the project folder**
```bash
   cd register-page
```

3. **Open in browser**
```bash
   open register.html
```
   or double-click `register.html`

> ⚠️ Make sure `assets/css/style.css` and `assets/img/register.png` paths are correct in your HTML file.
